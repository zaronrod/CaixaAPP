CaixaAPP - Aplicativo para controle de Fluxo de Caixa Familiar

Projeto solicitado para o Módulo Android Avançado - UTFRJ T25-2S

Instruções técnicas, compilação e ajustes:

- ATENÇÃO -
Se for executar no EMULADOR do ANDROID => Habilite a configuração abaixo pq usamos a autenticação do DEVICE real.

Configurando bloqueio de tela no Android Emulator

Com o emulador em execução, você verá a tela inicial padrão do Android ou o seu aplicativo.

Primeiro, vá para a tela inicial do emulador.
Em seguida, deslize de baixo para cima para abrir a lista de aplicativos instalados.
Localize e abra o aplicativo Settings (Configurações).

Acessando o menu de segurança

Dentro de Settings, role a tela até encontrar a opção Security ou Security & privacy.
O nome pode variar levemente de acordo com a versão do Android utilizada no emulador.

Definindo um bloqueio de tela

Entre no menu Security.
Procure a opção Screen lock.
O sistema pedirá que você escolha um tipo de bloqueio.
Selecione PIN ou Password.
Não escolha as opções Swipe ou None.

Siga as instruções exibidas na tela para configurar o PIN ou a senha.

Adicionando impressão digital (recomendado)

Após configurar o bloqueio de tela, a opção de impressão digital ficará disponível.

Volte ao menu Security.
Procure por Fingerprint, que pode estar dentro de um submenu chamado Biometrics.
Siga os passos indicados pelo sistema.

O emulador irá simular um sensor de impressão digital.
Para registrar a digital, utilize os controles estendidos do emulador, acessados pelos três pontos na lateral, e use a opção de simulação do sensor para “tocar” no leitor.

Finalização

Depois de concluir essas etapas no emulador, execute o aplicativo novamente.
O botão de login deverá acionar corretamente o prompt de autenticação.
